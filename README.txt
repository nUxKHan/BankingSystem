# Simple Banking System (Python)

A beginner-friendly command-line banking application built with Python and Object-Oriented Programming (OOP).

## Features

- Create a PIN
- Check account balance
- Deposit money
- Withdraw money
- PIN verification for security
- Menu-driven interface

## Technologies Used

- Python 3
- OOP (Classes and Objects)
